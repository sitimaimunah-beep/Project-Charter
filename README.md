# ACADORA

**Academic Operations & Data Resource Application**

## 📌 Deskripsi Proyek

ACADORA merupakan sistem berbasis web yang dirancang untuk mendukung dan mengelola layanan akademik mahasiswa secara terintegrasi. Sistem ini bertujuan untuk mempermudah proses akademik seperti pengisian KRS, evaluasi dosen, konsultasi akademik, pengajuan revisi nilai, serta akses terhadap hasil akademik.

## 🎯 Tujuan

* Menyediakan platform layanan akademik yang terintegrasi
* Meningkatkan efisiensi dalam pengelolaan aktivitas akademik
* Mempermudah interaksi antara mahasiswa dan pihak akademik

## ⚙️ Fitur Utama

* Pengisian KRS
* Evaluasi Dosen oleh Mahasiswa (EDOM)
* Konsultasi Bimbingan Akademik
* Pengajuan Revisi Nilai
* Bukti Tagihan Akademik
